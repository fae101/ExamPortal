System Features

User Authentication

Secure user registration and login system for both students and admins with proper access control.

Exam Management

Complete exam lifecycle management - create, update, delete exams with multiple questions.

Question Management

Add, update, and delete questions for each exam with flexible question formats.

Exam Taking

Students can take exams by answering questions with automatic submission and result calculation.

Results Viewing

Students can view their exam results, while admins can monitor all student performances.

Exam List

Display available exams for students to select and take at their convenience.
